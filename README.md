# Medical Billing Dashboard – Power BI

## Project Overview

This Power BI dashboard was developed to monitor healthcare revenue cycle performance and medical billing operations.

## Key Metrics

* Claim Status
* Denial Rate
* Coding Accuracy %
* Revenue Recovered
* ICD-10 Compliance %
* Resolution Rate
* High-Risk Coder Analysis

## Tools Used

* Power BI
* DAX
* Power Query
* SQL
* Excel

## Business Impact

* Identified denial trends and root causes.
* Improved visibility into coding quality.
* Supported revenue recovery initiatives.
* Automated manual reporting processes.

## Dashboard Features

* Interactive slicers
* Drill-through navigation
* KPI cards
* Conditional formatting
* Bookmarks
* Trend analysis

## Domain

Healthcare Revenue Cycle Management (RCM)
